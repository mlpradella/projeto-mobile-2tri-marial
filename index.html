Projeto escolar

O projeto foi feito com o intuito de melhorar a vida dos estudantes que tem dificuldades em acessar informações

Avisos

A página de avisos tem o objetivo de ajudar os estudantes que perdem os avisos dados pelo colégio. Alunos normalmente perdem os comunicados dados pelo colégio, então essa categoria tem o intuito de acabar com o este problema.

Avaliações

As avaliações tem o objetivo de deixar as datas, conteudos, materias das avaliações expostas. Como os avisos, o seu intuito é ajudar os alunos que se perdem nas datas de provas, ou atividades avaliativas.

Calendário

O calendario foi feito com o intuito dos alunos terem uma parte visual de datas comemorativas e avaliativas, sem que possam se esquecer delas ou perde-las. Os professores e administradores são quem fazem os avisos.

Lista

  • Login: ele ira garantir que cada aluno tenha acesso individual, protegendo dados pessoais e acadêmicos.
  • Avisos: é onde os professores e administradores postam os avisos para os estudantes.
  • Reclamação: página onde os alunos poderam postar sua opniões/reclamações, e os administradores acompanharam as reclamações publicadas
  • Calendario: a página de calendarios, é onde será possivel colocar as datas comemorativas e avaliativas para os alunos.
  • Avaliações: onde seram postadas pelos professores sobre as datas, conteudos, materias das datas avaliativas.
